Project Magpie

Project Magpie, AirBourne's revolutionary Remote Desktop Software, is poised to redefine the digital workspace experience. With key features such as on-device low-power rendering, high-security transfers utilizing AirBourne's RaE Security System, and a non-rigid release structure, Project Magpie prioritizes user-centric innovation. As it approaches the end of its beta phase, this cutting-edge software is preparing for a seamless transition to Dis//Connect. Beyond a mere name change, this evolution encompasses enhanced connectivity, innovative disconnection features, and an intuitive interface. Dis//Connect signifies the next phase in remote desktop experiences, where users can seamlessly navigate a customized workspace, even in offline mode. With a commitment to quality assured by beta testers and a vision that places users at its core, Project Magpie's transition to Dis//Connect promises to usher in a transformative era of unparalleled digital collaboration and access.
